## 📂 Project Unggulan

☕ **Coffee Store Performance Dashboard**  
Dashboard interaktif menggunakan Google Looker Studio untuk menganalisis performa penjualan coffee shop, tren penjualan, performa outlet, kategori produk, serta mendukung pengambilan keputusan bisnis.

📊 **Sales Performance Dashboard**  
Dashboard Microsoft Excel untuk memonitor penjualan, profit, segmentasi pelanggan, performa wilayah, dan tren penjualan.

📦 **Supply Chain Performance Dashboard**  
Dashboard Power BI untuk memonitor performa supply chain, keterlambatan pengiriman, profitabilitas produk, dan performa penjualan berdasarkan wilayah.

👥 **Customer Segmentation Dashboard (RFM & K-Means Clustering)**  
Proyek segmentasi pelanggan menggunakan Python dengan metode RFM Analysis dan K-Means Clustering untuk mendukung strategi pemasaran yang lebih tepat sasaran.

📑 **Invoice Verification & Billing Dashboard**  
Dashboard Microsoft Excel untuk proses verifikasi invoice, rekonsiliasi billing, monitoring KPI, dan evaluasi performa penagihan.

🗄️ **SQL Sales Analysis**  
Analisis data penjualan menggunakan SQL untuk menghasilkan insight bisnis melalui KPI, tren penjualan, analisis pelanggan, dan performa produk.
