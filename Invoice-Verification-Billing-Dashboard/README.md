# 📑 Invoice Verification & Billing Dashboard

## 📖 Project Overview

Proyek ini merupakan dashboard interaktif yang dikembangkan menggunakan **Microsoft Excel** untuk membantu proses verifikasi invoice dan monitoring performa billing. Seluruh proses dilakukan di Excel, mulai dari data cleaning, pencocokan data invoice dengan delivery order, analisis menggunakan Pivot Table, hingga pembuatan dashboard interaktif yang memudahkan proses monitoring dan pengambilan keputusan.

---

## 🛠️ Project Workflow

| Tahapan | Tools | Deskripsi |
|---------|-------|-----------|
| Data Cleaning | Microsoft Excel | Membersihkan data, memeriksa data duplikat, memastikan konsistensi format, serta menyiapkan data invoice dan delivery order untuk proses verifikasi. |
| Data Verification | Microsoft Excel | Melakukan pencocokan data invoice dengan delivery order menggunakan **XLOOKUP** dan **IF Formula** untuk menentukan status **MATCH** atau **MISMATCH**. |
| Data Analysis | Pivot Table | Menganalisis data billing untuk menghitung KPI, invoice yang belum diproses, tren billing bulanan, serta invoice dengan nilai tertinggi. |
| Dashboard Development | Microsoft Excel | Membangun dashboard interaktif menggunakan **Pivot Chart**, **Slicer**, dan KPI Cards sehingga proses monitoring billing menjadi lebih mudah dan efisien. |

---

## 📌 Business Problem

Proses verifikasi invoice yang masih dilakukan secara manual sering kali membutuhkan waktu yang lama dan berisiko menimbulkan kesalahan pencatatan. Tanpa sistem monitoring yang terintegrasi, perusahaan akan mengalami beberapa kendala, seperti:

- Ketidaksesuaian antara data invoice dan delivery order.
- Sulit memantau invoice yang masih pending.
- Proses rekonsiliasi invoice menjadi kurang efisien.
- Terbatasnya informasi mengenai performa billing secara keseluruhan.

---

## 🎯 Project Objectives

Proyek ini dikembangkan untuk:

- Memverifikasi kesesuaian data invoice dengan delivery order.
- Mengidentifikasi invoice yang **MATCH** maupun **MISMATCH** secara otomatis.
- Memonitor performa billing melalui KPI dan visualisasi interaktif.
- Mempermudah proses rekonsiliasi invoice.
- Mendukung pengambilan keputusan yang lebih cepat dan akurat.

---

## 📊 Dashboard Features

Dashboard menyediakan berbagai informasi penting, antara lain:

- Total Invoice
- Total Billing Amount
- Pending Invoice
- Match Rate (%)
- Status MATCH / MISMATCH
- Invoice Reconciliation
- Delivery Order Validation
- Monthly Billing Trend
- Top 5 Highest Invoice Amount
- Billing Performance Overview

---

## 📈 Key Insights

- Dashboard mencatat **300 invoice** dengan total nilai billing sebesar **Rp 1.875.075.000**.
- Sebanyak **282 invoice (94%)** berhasil terverifikasi (**MATCH**) dengan delivery order, sedangkan **18 invoice (6%)** masih berstatus **MISMATCH**.
- Invoice bernilai tertinggi adalah **INV0001** dengan total **Rp 20.000.000**, sehingga menjadi prioritas untuk proses monitoring.
- Tren billing bulanan menunjukkan fluktuasi nilai penagihan sepanjang tahun sehingga dapat digunakan untuk mengevaluasi performa billing setiap periode.
- Dashboard membantu mempercepat proses rekonsiliasi invoice dan mempermudah identifikasi invoice yang memerlukan tindak lanjut.

---

## 💡 Business Recommendations

- Melakukan verifikasi invoice secara berkala untuk meminimalkan kesalahan pencatatan.
- Memprioritaskan penyelesaian invoice dengan status **MISMATCH** agar tidak menghambat proses pembayaran.
- Menggunakan dashboard sebagai alat monitoring rutin bagi tim Finance dan Accounting.
- Memanfaatkan analisis tren billing untuk mengevaluasi performa proses penagihan setiap periode.
- Mengembangkan dashboard dengan pembaruan data secara berkala agar informasi selalu relevan.
