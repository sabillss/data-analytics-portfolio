# 📑 Invoice Verification & Billing Dashboard

An interactive **Microsoft Excel dashboard** developed to verify invoice accuracy, reconcile billing data with delivery records, and monitor billing performance through KPI tracking and dynamic visualizations.

---

## 📌 Project Overview

Invoice verification is an essential process to ensure billing accuracy and maintain healthy cash flow. This project automates invoice reconciliation by matching invoice records with delivery orders, identifying discrepancies, and providing an interactive dashboard for billing performance monitoring.

The dashboard enables finance and operations teams to quickly validate invoices, track outstanding bills, and analyze billing trends for better decision-making.

---

## 🎯 Business Problem

Manual invoice verification is time-consuming and prone to human error. Without an automated monitoring system, businesses may experience:

- Invoice mismatches between billing and delivery records
- Delayed invoice verification
- Difficulties tracking pending invoices
- Limited visibility into billing performance

These challenges can affect operational efficiency and cash flow management.

---

## 💡 Solution

An interactive dashboard was built in **Microsoft Excel** to automate invoice verification and monitor billing performance.

The project includes:

- Data cleaning and preparation
- Invoice verification and reconciliation
- Data matching using **XLOOKUP**
- Validation using **IF Formula**
- Data analysis with **Pivot Tables**
- Interactive dashboard development using **Pivot Charts**
- Billing performance monitoring

## 📈 Key Insights

- Automatically identifies invoices that match or mismatch with delivery records.
- Monitors pending invoices to improve billing follow-up.
- Displays monthly billing trends for financial performance analysis.
- Highlights high-value invoices that require priority monitoring.
- Improves invoice reconciliation efficiency and reduces manual verification effort.
