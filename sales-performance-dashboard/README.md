# 📊 Sales Performance Dashboard

An interactive **Microsoft Excel dashboard** developed to monitor sales performance, profitability, customer segmentation, and regional sales trends using the **Superstore Dataset**.

---

## 📌 Project Overview

This project analyzes retail sales data to provide insights into business performance through an interactive Excel dashboard. By utilizing Pivot Tables, Pivot Charts, KPI Cards, and Slicers, the dashboard enables users to monitor key business metrics and identify sales opportunities efficiently.

---

## 🎯 Business Problem

The company does not have a centralized dashboard to effectively monitor:

- Sales performance
- Profitability
- Customer segmentation
- Regional sales trends

As a result, management faces challenges in evaluating business performance and making timely, data-driven decisions.

---

## 💡 Solution

An interactive dashboard was built using **Microsoft Excel** to transform raw sales data into meaningful business insights.

The project includes:

- Data cleaning and preprocessing
- Data analysis using Pivot Tables
- Interactive dashboard development
- Dynamic filtering using Slicers

The dashboard focuses on monitoring:

- Sales Performance
- Profitability
- Customer Segmentation
- Regional Sales Performance
- Monthly Sales Trends

---

## 🛠 Tools & Technologies

- Microsoft Excel

## 📊 Dashboard Features

### KPI Cards

- 💰 Total Sales
- 📈 Total Profit
- 🛒 Total Orders
- 📊 Profit Margin

### Visualizations

- Monthly Sales Trend
- Sales by Category
- Profit by Region
- Customer Segment Distribution

### Interactive Filters

- Region
- Category
- Customer Segment
- Order Date

---

## 📈 Key Insights

- **Technology** category has the highest profit margin.
- **West** region generates the highest profit.
- **Consumer** segment contributes the largest share of total sales.
- Monthly sales trends show consistent growth over time.
