# 👥 Customer Segmentation Dashboard (RFM Analysis + K-Means Clustering)

## 📖 Project Overview

Proyek ini bertujuan membangun dashboard interaktif untuk melakukan segmentasi pelanggan menggunakan metode **RFM (Recency, Frequency, Monetary) Analysis** dan **K-Means Clustering**. Seluruh proses dikerjakan menggunakan Python, mulai dari data cleaning, feature engineering, machine learning, hingga visualisasi data. Dashboard kemudian dipublikasikan melalui **GitHub Pages** sehingga dapat diakses secara online dan mendukung pengambilan keputusan bisnis berbasis data.

---

## 🛠️ Project Workflow

| Tahapan | Tools | Deskripsi |
|---------|-------|-----------|
| Data Cleaning | Python (Pandas, NumPy) | Membersihkan data transaksi, menangani missing values dan data duplikat, memperbaiki tipe data, serta menyiapkan dataset untuk proses analisis. |
| Feature Engineering | Python | Menghitung nilai **Recency**, **Frequency**, dan **Monetary (RFM)** sebagai dasar segmentasi pelanggan. |
| Machine Learning | Scikit-learn | Melakukan segmentasi pelanggan menggunakan algoritma **K-Means Clustering** untuk mengelompokkan pelanggan berdasarkan karakteristik transaksi. |
| Data Visualization | Plotly | Membuat visualisasi interaktif untuk menampilkan distribusi pelanggan, analisis RFM, serta hasil clustering. |
| Dashboard Development | Python, HTML, CSS | Mengembangkan dashboard interaktif yang menyajikan hasil analisis, visualisasi, serta insight bisnis dalam satu tampilan. |
| Deployment | GitHub Pages | Mempublikasikan dashboard sehingga dapat diakses secara online melalui browser. |

---

## 📌 Business Problem

Banyak perusahaan telah mengumpulkan data transaksi pelanggan, namun belum memanfaatkannya secara optimal untuk memahami perilaku pelanggan. Kondisi tersebut menyebabkan beberapa permasalahan, antara lain:

- Kampanye pemasaran kurang tepat sasaran.
- Pelanggan bernilai tinggi sulit diidentifikasi.
- Strategi retensi pelanggan belum optimal.
- Potensi bisnis dari analisis perilaku pelanggan sering kali belum dimanfaatkan secara maksimal.

---

## 🎯 Project Objectives

Proyek ini dikembangkan untuk:

- Melakukan segmentasi pelanggan berdasarkan metode **RFM Analysis**.
- Mengelompokkan pelanggan menggunakan algoritma **K-Means Clustering**.
- Memvisualisasikan hasil segmentasi dalam dashboard interaktif.
- Membantu bisnis memahami karakteristik pelanggan berdasarkan perilaku transaksi.
- Mendukung penyusunan strategi pemasaran dan retensi pelanggan berbasis data.

---

## 📊 Dashboard Features

Dashboard menyediakan berbagai analisis, antara lain:

- Customer Analytics
- RFM Analysis (Recency, Frequency, Monetary)
- Customer Segmentation
- K-Means Cluster Visualization
- Revenue Distribution Analysis
- Customer Distribution Visualization
- Cluster Summary
- Customer Profile Analysis
- Business Insights
- Marketing Recommendations

---

## 📈 Key Insights

Hasil analisis berhasil mengelompokkan pelanggan ke dalam **empat segmen utama**, yaitu:

- ⭐ **High-Value / VIP Customers** → pelanggan dengan nilai transaksi tinggi dan loyalitas terbaik.
- 💎 **Loyal Mid-Value Customers** → pelanggan aktif dengan frekuensi transaksi yang cukup tinggi.
- 🛍️ **Occasional Low-Value Customers** → pelanggan yang bertransaksi sesekali dengan kontribusi revenue relatif rendah.
- ⚠️ **Lost / Dormant Customers** → pelanggan yang sudah lama tidak melakukan transaksi dan berpotensi churn.

Dashboard memungkinkan pengguna mengeksplorasi karakteristik setiap segmen pelanggan, distribusi revenue, serta pola perilaku pelanggan melalui visualisasi interaktif.
